Q：什麼是經鼻胃鏡？

從鼻子進去的細胃鏡

一種刺激感更低、恢復更快的現代內視鏡檢查方式

簡單一句話

經鼻胃鏡是將一條直徑約 0.5 公分的細管內視鏡，從鼻腔進入，經過食道到達胃部與十二指腸進行檢查，因為不經過舌根與軟顎，整體刺激感比傳統口胃鏡低。

👃

從鼻腔進入

鏡子通過鼻腔、咽喉進入食道與胃部，繞過口腔舌根，大幅降低噁心嘔吐反射。

📏

鏡子極細 0.5cm

直徑僅約 0.5 公分，比傳統胃鏡細許多，通過鼻腔時異物感相對較低。

📷

高清影像解析

現代經鼻胃鏡影像解析度大幅提升，同樣可以清楚觀察食道、胃與十二指腸狀況。

⚡

快速恢復日常

無需鎮靜麻醉，檢查後可立即恢復正常活動，不需要等待麻醉退效。

Q：做經鼻胃鏡會不會痛？

多數是異物感，不是疼痛

通常不會出現劇烈疼痛，大多數人都能順利完成

常見感受

1

鼻子有點痠脹感

鏡子通過鼻腔時，可能感覺輕微痠脹，通常在幾秒內適應。

2

喉嚨有異物感

鏡子經過喉嚨時會有異物感，但因為不經舌根，通常比口胃鏡舒適許多。

3

胃部有輕微脹感

檢查時會注入少量氣體幫助觀察，可能感到輕微脹氣，屬於正常現象。

醫療減痛措施

💊
局部麻醉噴霧

檢查前醫師會在鼻腔與喉嚨噴上局部麻醉藥，有效減少不適感，幫助您順利完成檢查。

⚠️
如感到特別不適

請隨時舉手示意醫護人員，不必忍耐。醫師會視情況調整或暫停檢查。

適合對象

哪些人適合做經鼻胃鏡？

怕噁心、怕麻醉的人，是很好的選擇

✅ 特別適合的族群
  • 做一般胃鏡很容易噁心嘔吐的人
  • 害怕做傳統胃鏡、存在心理恐懼的人
  • 不想使用鎮靜麻醉、希望保持清醒的人
  • 希望檢查後可以立即恢復日常活動的人
  • 需要定期追蹤胃部狀況的慢性患者
  • 駕車、需要當天立即出行者
⛔ 可能不適合的族群
  • 嚴重鼻中隔彎曲的人
  • 鼻腔嚴重狹窄的人
  • 鼻甲腫大影響通道的人
  • 本身容易流鼻血的人
  • 近期有鼻腔手術史者
🩺

醫師會在檢查前先評估鼻腔狀況，再決定最合適的檢查方式，請放心諮詢。

術前準備

檢查前需要注意什麼？

提前做好準備，讓檢查過程更順利

🍽️ 飲食準備

  • ✓ 檢查前 8 小時禁食：前一晚 24:00 後不可進食固體食物，確保胃部清空。
  • ✓ 可少量喝清水：可適量飲水至檢查前 2–4 小時（依醫院規定）。
  • ✓ 禁止牛奶、果汁、咖啡：有色或含奶飲品可能影響視野。

💊 用藥告知

  • ✓ 告知醫師正在服用的所有藥物，尤其是抗凝血藥（如阿斯匹靈、Warfarin）。
  • ✓ 告知有無藥物過敏史，特別是局部麻醉藥過敏。
  • ✓ 告知有無鼻部相關疾病或過往手術史。

📋 報到準備

  • ✓ 依預約時間提早 15 分鐘到院，完成報到手續。
  • ✓ 攜帶健保卡、身分證件與轉診單（如有）。
  • ✓ 穿著舒適寬鬆衣物，避免高領或緊身衣。
  • ✓ 取下假牙、活動牙橋（如有）。
⏰
禁食時間重要提醒

若未確實禁食，可能需要改期，以確保您的安全與檢查效果。請務必遵守禁食規定。

檢查流程

當天檢查怎麼進行？

完整了解流程，減少未知的緊張感

📋

第一步：報到與確認

到達後完成掛號報到，護理師確認禁食狀況、用藥史及健康問卷，簽署同意書。

💊

第二步：鼻腔局部麻醉

護理師或醫師在鼻腔噴入局部麻醉藥，等待約 5–10 分鐘使麻醉生效，減少進鏡時的不適感。

🛏️

第三步：擺位準備

側躺或採半臥姿勢，放鬆身體。醫師會在旁說明進鏡步驟，引導您配合呼吸。

🔬

第四步：進行檢查

鏡子從鼻腔緩緩進入，醫師仔細觀察食道、胃及十二指腸。整個過程約 5–15 分鐘，必要時可進行組織切片。

✅

第五步：結束與說明

鏡子緩緩退出，醫師說明初步檢查結果，若有切片需等待 1–2 週病理報告。

🏃

第六步：術後恢復

因無需鎮靜麻醉，可立即離開。短暫休息後即可恢復日常活動，當天可正常駕車。

術後照護

檢查後需要注意什麼？

了解術後正常反應與需要回診的警訊

✅ 正常術後反應

  • ✓ 鼻腔輕微不適或少量鼻血（少見，通常短暫）
  • ✓ 喉嚨些微不舒服感（通常數小時內緩解）
  • ✓ 輕微脹氣或打嗝（氣體排出後自然緩解）
  • ✓ 麻醉退效前暫時少量進食（約 30 分鐘後可進食）

⚠️ 需要回診的警訊

🚨
出現以下狀況請立即就醫

• 持續或大量鼻出血無法止住
• 劇烈胸痛或腹痛
• 吞嚥困難或有哽塞感
• 發燒（體溫超過 38°C）
• 嘔血或黑色糞便

由於不需要鎮靜麻醉，檢查後可自行離開，通常不需要特別安排家人陪同。但若是長者、體力較弱或心理上較緊張的患者，建議家人陪同以增加安心感。

實際進鏡檢查約 5–15 分鐘，加上報到、麻醉準備與術後說明，整體在院時間約 30–60 分鐘。若有進行切片或特殊處置，可能稍長。

經鼻胃鏡因為設計較細緻，通常院所會酌收自費費用，費用詳情建議直接向診所或醫院確認，以取得最準確資訊。

組織切片的病理報告通常需要 1–2 週。結果出來後，醫師會通知您回診說明，或透過診所客服聯繫方式告知。請記得預留回診時間，不建議自行推測結果。
